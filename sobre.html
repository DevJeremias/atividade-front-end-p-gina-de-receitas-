Sobre o Site "Receitas Deliciosas"

Olá, eu sou Henrique Jeremias e vou explicar como eu desenvolvi o site "Receitas Deliciosas". Este projeto foi criado com o objetivo de apresentar um site funcional e visualmente atraente para compartilhar receitas.

1. Estrutura HTML

O site é composto por três páginas principais:

  • Página Inicial: Exibe receitas em destaque e outras receitas.
  • Sobre: Fornece informações sobre o site e o desenvolvedor.
  • Contato: Oferece um link para meu perfil no GitHub para contato.

Cada página segue uma estrutura HTML básica, incluindo um <header> com um título e uma barra de busca, uma <nav> para navegação entre as páginas, um <main> para o conteúdo principal e um <footer> com informações de direitos autorais.

2. Design e Layout

O design do site é moderno e limpo, utilizando CSS para garantir uma aparência atraente:

  • Header: Fundo coral com texto branco e barra de busca centralizada.
  • Nav: Menu de navegação com fundo escuro e texto branco, que muda de cor ao passar o mouse.
  • Conteúdo Principal: Seções organizadas com cartões de receita e uma grade para facilitar a visualização.
  • Footer: Fundo escuro com texto branco, proporcionando uma conclusão visual limpa.

3. Funcionalidade da Barra de Busca

A barra de busca utiliza JavaScript para filtrar receitas com base no texto inserido:

  • HTML: Cada cartão de receita tem um atributo data-tags com palavras-chave.
  • JavaScript: O script filtra os cartões de receita para mostrar apenas os que correspondem à busca. Também permite buscar ao pressionar Enter.

4. Estilo e Responsividade

A folha de estilos CSS garante:

  • Fonte e Cores: Fontes modernas e cores que criam um contraste agradável e mantêm a legibilidade.
  • Layout Responsivo: Design ajustável para diferentes tamanhos de tela.

5. Considerações Finais

O site "Receitas Deliciosas" foi desenvolvido com simplicidade e funcionalidade em mente. O HTML, CSS e JavaScript proporcionam uma experiência de usuário agradável e uma interface intuitiva. Espero que você goste do site e encontre muitas receitas deliciosas!
